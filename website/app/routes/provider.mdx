# Toast Provider

The `<ToastProvider />` component is used to wrap your application and provide the context for the toasts. It accepts the following props:

## Position

You can customize the position of the toasts by using the `position` prop of the `ToastProvider` component:

<Positions />

## Theme

You can set the theme of the toasts using the `theme` prop, which accepts the values: `light`, `dark`, and `system`.

```jsx
<ToastProvider theme="light">
  <App />
</ToastProvider>
```

## MaxToasts

By default, the maximum number of toasts is set to `4`. You can change this value using the `maxToasts` prop:

```jsx
<ToastProvider maxToasts={8}>
  <App />
</ToastProvider>
```

## ToastFont

By default, the font of the toasts is:

```txt
ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, Segoe
UI, Roboto, Helvetica Neue, Arial, Noto Sans, sans-serif, Apple Color Emoji,
Segoe UI Emoji, Segoe UI Symbol, Noto Color Emoji;
```

You can change this value using the `toastFont` prop. In this case, we are using the utility class [`font-sans` from Tailwind CSS](https://tailwindcss.com/docs/font-family#setting-the-font-family) to set the font:

```jsx
<ToastProvider toastFont="font-sans">
  <App />
</ToastProvider>
```
